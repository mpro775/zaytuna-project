# أيقونات التطبيق

يجب إضافة الأيقونات التالية لهذا التطبيق:

## أحجام مطلوبة للـ PWA:

- `icon-72x72.png` - 72x72px
- `icon-96x96.png` - 96x96px
- `icon-128x128.png` - 128x128px
- `icon-144x144.png` - 144x144px
- `icon-152x152.png` - 152x152px
- `icon-192x192.png` - 192x192px
- `icon-384x384.png` - 384x384px
- `icon-512x512.png` - 512x512px

## مواصفات التصميم:

- خلفية: أخضر زيتوني (#2e7d32)
- شعار: زيتون أو شجرة زيتون
- تنسيق: PNG مع خلفية شفافة
- دائرية أو مربعة مع حواف مستديرة

## أدوات إنشاء الأيقونات:

يمكن استخدام أدوات مثل:
- Figma
- Adobe Illustrator
- Canva
- أو خدمات عبر الإنترنت مثل favicon.io

## ملاحظات:

- تأكد من أن الأيقونات واضحة عند جميع الأحجام
- استخدم نفس التصميم لجميع الأحجام مع تعديل الدقة فقط
- الأيقونات يجب أن تكون بصيغة PNG-24 مع خلفية شفافة
