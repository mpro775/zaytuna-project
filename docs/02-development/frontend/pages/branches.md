# صفحة إدارة الفروع (Branches)

## نظرة عامة

صفحة إدارة الفروع تسمح للمستخدمين بإدارة فروعهم.

**الموقع**: `src/pages/Branches/Branches.tsx`  
**المسار**: `/branches`  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

- عرض قائمة الفروع
- إضافة/تعديل/حذف فرع
- إدارة المخازن لكل فرع
- تبديل الفرع النشط

---

## APIs المستخدمة

- `GET /branches` - جلب الفروع
- `POST /branches` - إضافة فرع
- `PATCH /branches/:id` - تحديث فرع
- `DELETE /branches/:id` - حذف فرع
