# صفحة 404 (Not Found)

## نظرة عامة

صفحة 404 تظهر عندما يحاول المستخدم الوصول لصفحة غير موجودة.

**الموقع**: `src/pages/NotFound/NotFound.tsx`  
**المسار**: `*` (أي مسار غير موجود)  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

- عرض رسالة خطأ واضحة
- زر للعودة للصفحة الرئيسية
- رابط للصفحة السابقة

---

## المكونات المستخدمة

- `Button`: زر العودة
- `Typography`: النصوص

---

## تدفق المستخدم (User Flow)

1. المستخدم يحاول الوصول لصفحة غير موجودة
2. يرى صفحة 404
3. يمكنه:
   - العودة للصفحة الرئيسية
   - استخدام زر "رجوع" في المتصفح
