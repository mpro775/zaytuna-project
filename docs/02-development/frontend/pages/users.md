# صفحة إدارة المستخدمين (Users)

## نظرة عامة

صفحة إدارة المستخدمين تسمح للمسؤولين بإدارة مستخدمي النظام.

**الموقع**: `src/pages/Users/UsersList.tsx`  
**المسار**: `/users` (سيتم إضافته)  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

- عرض قائمة المستخدمين
- إضافة/تعديل/حذف مستخدم
- إدارة الأدوار والصلاحيات
- تغيير كلمة المرور
- تفعيل/تعطيل المستخدم

---

## APIs المستخدمة

- `GET /users` - جلب المستخدمين
- `POST /users` - إضافة مستخدم
- `PATCH /users/:id` - تحديث مستخدم
- `DELETE /users/:id` - حذف مستخدم
- `PATCH /users/:id/password` - تغيير كلمة المرور
