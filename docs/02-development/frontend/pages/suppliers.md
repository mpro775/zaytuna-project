# صفحة إدارة الموردين (Suppliers)

## نظرة عامة

صفحة إدارة الموردين تسمح للمستخدمين بإدارة علاقاتهم مع الموردين.

**الموقع**: `src/pages/Suppliers/Suppliers.tsx`  
**المسار**: `/suppliers`  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

- عرض قائمة الموردين
- إضافة/تعديل/حذف مورد
- تتبع الأرصدة المستحقة
- سجل المشتريات

---

## APIs المستخدمة

- `GET /suppliers` - جلب الموردين
- `POST /suppliers` - إضافة مورد
- `PATCH /suppliers/:id` - تحديث مورد
- `DELETE /suppliers/:id` - حذف مورد
