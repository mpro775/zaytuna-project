# صفحة الإعدادات (Settings)

## نظرة عامة

صفحة الإعدادات تسمح للمستخدمين بتكوين إعدادات التطبيق.

**الموقع**: `src/pages/Settings/Settings.tsx`  
**المسار**: `/settings`  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

### 1. إعدادات الشركة
- معلومات الشركة
- الشعار
- الإعدادات الإقليمية

### 2. إعدادات النظام
- إدارة الكاش
- النسخ الاحتياطي
- المراقبة

### 3. إعدادات الأمان
- سياسات كلمات المرور
- إدارة الجلسات
- المصادقة الثنائية

### 4. إعدادات النسخ الاحتياطي
- التكوين التلقائي
- السجل
- التحميل

---

## APIs المستخدمة

- `GET /settings` - جلب الإعدادات
- `PATCH /settings` - تحديث الإعدادات
- `POST /settings/backup` - إنشاء نسخة احتياطية
