# صفحة تسجيل الدخول (Login)

## نظرة عامة

صفحة تسجيل الدخول هي نقطة الدخول للتطبيق.

**الموقع**: `src/pages/Login/Login.tsx`  
**المسار**: `/login`  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

### 1. تسجيل الدخول
- إدخال اسم المستخدم/البريد الإلكتروني
- إدخال كلمة المرور
- تسجيل الدخول

### 2. المصادقة الثنائية (2FA)
- إدخال رمز التحقق
- إرسال رمز جديد
- اختيار طريقة الإرسال (SMS/Email)

### 3. خيارات إضافية
- تذكر الجهاز
- نسيت كلمة المرور
- رابط التسجيل

---

## المكونات المستخدمة

- `Input`: حقول الإدخال
- `Button`: زر تسجيل الدخول
- `Modal`: نافذة 2FA

---

## تدفق المستخدم (User Flow)

1. المستخدم يفتح `/login`
2. يدخل بيانات تسجيل الدخول
3. يضغط "تسجيل الدخول"
4. إذا كان 2FA مفعل:
   - يظهر نموذج إدخال الرمز
   - يدخل الرمز
5. يتم توجيهه للـ Dashboard

---

## APIs المستخدمة

- `POST /auth/login` - تسجيل الدخول
- `POST /auth/verify-2fa` - التحقق من 2FA
- `POST /auth/send-2fa-code` - إرسال رمز 2FA

---

## حالات الاستخدام

### حالة الاستخدام 1: تسجيل دخول عادي
المستخدم يسجل دخول بدون 2FA:
1. يدخل البيانات
2. يضغط "تسجيل الدخول"
3. يتم توجيهه للـ Dashboard

### حالة الاستخدام 2: تسجيل دخول مع 2FA
المستخدم يسجل دخول مع 2FA:
1. يدخل البيانات
2. يضغط "تسجيل الدخول"
3. يظهر نموذج 2FA
4. يدخل الرمز
5. يتم توجيهه للـ Dashboard
