# صفحة إدارة المخازن (Warehouses)

## نظرة عامة

صفحة إدارة المخازن تسمح للمستخدمين بإدارة مخازنهم.

**الموقع**: `src/pages/Warehouses/Warehouses.tsx`  
**المسار**: `/warehouses`  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

- عرض قائمة المخازن
- إضافة/تعديل/حذف مخزن
- ربط المخزن بفرع
- عرض المخزون لكل مخزن

---

## APIs المستخدمة

- `GET /warehouses` - جلب المخازن
- `POST /warehouses` - إضافة مخزن
- `PATCH /warehouses/:id` - تحديث مخزن
- `DELETE /warehouses/:id` - حذف مخزن
