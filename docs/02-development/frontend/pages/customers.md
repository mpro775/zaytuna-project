# صفحة إدارة العملاء (Customers)

## نظرة عامة

صفحة إدارة العملاء تسمح للمستخدمين بإدارة علاقاتهم مع العملاء: إضافة، تعديل، حذف، وعرض تفاصيل العملاء.

**الموقع**: `src/pages/Customers/Customers.tsx`  
**المسار**: `/customers`  
**الحالة**: ✅ مكتملة

---

## الوظائف الرئيسية

### 1. عرض العملاء
- جدول العملاء
- البحث والفلترة
- ترقيم الصفحات

### 2. إدارة العملاء
- إضافة عميل جديد
- تعديل عميل
- حذف عميل
- عرض تفاصيل العميل

### 3. معلومات إضافية
- سجل المبيعات
- الأرصدة المستحقة
- نقاط الولاء

---

## APIs المستخدمة

- `GET /customers` - جلب العملاء
- `POST /customers` - إضافة عميل
- `PATCH /customers/:id` - تحديث عميل
- `DELETE /customers/:id` - حذف عميل
